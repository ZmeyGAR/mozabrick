Wie es gemacht wird

Sie haben noch kein Fotomosaik? Dann kaufen Sie jetzt!

  • Kaufen Sie einen QBRIX Set

    Sie kaufen einen QBRIX Set in der Größe 44x44cm, die Teile aus fünf Farben enthalten

  • Laden Sie ein Foto hoch

    Besuchen Sie qbrix.com Webseite, Geben Sie den Zugangscode ein, der in der in der Setanleitung genannt wurde. Laden Sie das Foto, dass Sie zusammensetzen möchten.

  • Erhalten Sie eine Anleitung

    Sie erhalten sofort eine kostenlose PDF-Bauanleitung zum Zusammensetzen Ihres Bildes per E-Mail.

  • Bauen Sie ein Mosaik

    Setzen Sie Ihr Motiv für sich selbst oder als Geschenk zusammen und genießen Sie den kreativen Prozess

  • Immer wieder!

    Gefällt Ihnen Ihr bild nicht mehr, dann können Sie jederzeit ein neues Bild zusammensetzen!
